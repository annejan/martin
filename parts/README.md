# parts/ — the reusable part library

Named, tested show fragments that productions compose from — the real capital that makes every next
production cheaper. A part earns its place here by being **proven in a production** first (built,
rendered, approved in a showbook), then extracted with a neutral name and a note on how to use it.

Conventions (to settle as the first parts land):
- one file per part, `.seq`/`.show`-snippet syntax, with a header comment: what it looks like,
  what it needs (assets/uniforms), and which production proved it;
- neutral names — `fireflies-condense`, `wall-tunnel`, `explode-to-capture` — never theme names.

Empty until the camping showbook's first scenes graduate from ▣ gebouwd to ★ goedgekeurd.
